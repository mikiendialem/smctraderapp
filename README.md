SmartMoney Journal

SmartMoney Journal is an AI-powered trading journal and performance analytics application built for forex and crypto traders using Smart Money Concepts (SMC).

The app helps traders:
- Track trades professionally
- Analyze performance
- Improve discipline
- Monitor psychology
- Review risk management
- Generate AI-powered trading insights

---

Features

# Trading Journal
- Add and manage trades
- Upload chart screenshots
- Track entries, stop loss, and take profit
- Risk-to-reward calculations
- Trade notes and emotional tracking

# Smart Money Concepts Support
- BOS
- CHoCH
- Order Blocks
- Fair Value Gaps (FVG)
- Liquidity Sweeps
- OTE setups
- Market Structure analysis

# Analytics Dashboard
- Win rate tracking
- Profit/Loss overview
- Performance charts
- Trading session analytics
- Monthly growth tracking
- Discipline and consistency scoring

# Authentication
- Secure login/signup
- Google authentication
- Role-based access control
- Protected dashboards

# AI Features
- AI trading feedback
- Pattern analysis
- Weekly performance summaries
- Trading behavior insights

Getting Started

# Prerequisites

Before running the project, install:

- Android Studio
- Android SDK
- Kotlin Plugin

---

Local Development

Clone the repository:

```bash
git clone <your-repository-url>
cd smartmoney-journal
````

Create a `.env` file in the project directory:

```env
GEMINI_API_KEY=your_api_key_here
```

Sync Gradle dependencies and run the project inside Android Studio.

---

# Running the App

1. Open Android Studio
2. Select **Open Project**
3. Choose the SmartMoney Journal project folder
4. Allow Gradle sync and dependency installation
5. Connect an Android device or start an emulator
6. Run the application
